## 1.目标

## 2.难度

低

## 3.环境要求

- 系统：Windows、Linux、MacOS
- IntelliJ IDEA（可选）


## 4.项目介绍

本次项目中, 将开发一个 hello_world.html 页面, 引入 nanhu-print-js 框架的 js 文件, 模拟真实环境下, 使用 nanhu-print-js 框架进行开发的过程。

其代码结构为:

```
│  hello_world.html
│
└─js
    ├─jsonix
    │      Jsonix-all.js
    │
    └─nanhuprint
            interpreter.js
            nanhuprint.js
            nanhuprintEval.js
```
